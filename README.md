# PCGL @ CNU — Official Website

Plant Computational Genomics Lab, Chungnam National University

## File Structure
```
pcglcnu.github.io/
├── index.html
├── css/style.css
├── js/main.js
├── assets/            ← 이미지 파일을 여기에 넣으세요
│   ├── hero-bg.jpg    ← 히어로 배경 이미지
│   ├── cnu-logo.png   ← 충남대 로고
│   └── professor.jpg  ← 교수님 사진
└── pages/
    ├── news.html
    ├── professor.html
    ├── members.html
    ├── publications.html
    ├── research.html
    ├── events.html
    └── opportunities.html
```

## 배포 방법
1. GitHub에서 `pcglcnu.github.io` 저장소 생성
2. 이 폴더의 모든 파일을 저장소에 업로드
3. Settings → Pages → Branch: main → Save
